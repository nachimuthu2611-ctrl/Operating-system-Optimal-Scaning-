# Disk Scheduling Algorithm Simulator

## Overview
This project simulates disk scheduling algorithms used in Operating Systems.

Algorithms implemented:
- FCFS (First Come First Serve)
- SSTF (Shortest Seek Time First)
- SCAN (Elevator Algorithm)

---

## Objectives
- Understand disk scheduling algorithms
- Compare performance
- Calculate seek time
- Visualize head movement

---

## Technologies Used
- C Programming
- GCC Compiler

---

## How to Run

Compile:
gcc main.c -o disk

Run:
./disk

---

## Input Example
Enter number of requests: 5
Enter requests: 98 183 37 122 14
Enter initial head position: 53
Enter disk size: 200

---

## Output Example
Total Seek Time (FCFS): 640
Total Seek Time (SSTF): 236
Total Seek Time (SCAN): 208
